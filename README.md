## 🎓 عن المعهد

**المعهد المصري لأكاديمية الإسكندرية للإدارة والمحاسبة** هو واحد من أرقى المؤسسات التعليمية في مصر، تأسس سنة 1996 بالقرار الوزاري رقم 1540. من 60 طالب في البداية وصلنا لأكتر من 4000 طالب وطالبة دلوقتي!

### 🌟 رؤيتنا
نكون المعهد الرائد في تخريج كوادر مؤهلة قادرة على المنافسة في سوق العمل المحلي والإقليمي، ونساهم في التنمية الاقتصادية والاجتماعية.

### 🎯 رسالتنا
تقديم تعليم عالي الجودة في مجالات الإدارة والمحاسبة وتكنولوجيا المعلومات، وإعداد خريجين مؤهلين بالمهارات اللازمة لسوق العمل.

## 📚 التخصصات المتاحة

### 💼 إدارة الأعمال
- **الوصف**: هتتعلم إزاي تدير الشركات والمشاريع زي المحترفين
- **مدة الدراسة**: 4 سنوات
- **فرص العمل**: مجال واسع وفرص شغل كتير في كل الشركات
- **المهارات**: القيادة، التخطيط، التسويق، إدارة الموارد البشرية

### 📊 المحاسبة والمراجعة
- **الوصف**: خليك محاسب محترف واشتغل في أكبر الشركات
- **مدة الدراسة**: 4 سنوات
- **فرص العمل**: تخصص مطلوب في كل مكان ومضمون
- **المهارات**: إعداد القوائم المالية، المراجعة، المحاسبة الضريبية

### 💻 نظم معلومات الأعمال
- **الوصف**: تخصص المستقبل اللي بيربط بين التكنولوجيا والإدارة
- **مدة الدراسة**: 4 سنوات
- **فرص العمل**: مجال واسع جداً ومطلوب في عصر التكنولوجيا
- **المهارات**: تطوير الأنظمة، تحليل البيانات، إدارة قواعد البيانات

## 🚀 التقنيات المستخدمة

## 🔧 تشغيل الموقع محلياً

### 📋 المتطلبات
- Web Server (Apache/Nginx)
- PHP 7.4+
- MySQL 5.7+
- Git

### 🚀 خطوات التشغيل

```bash
# 1. استنساخ المشروع
git clone https://github.com/obieda-hussien/EIA.git

# 2. الانتقال لمجلد المشروع
cd EIA

# 3. تثبيت المتطلبات (اختياري للتطوير)
npm install

# 4. تشغيل خادم محلي
npm start
# أو
python -m http.server 8000
# أو
php -S localhost:8000

# 5. فتح المتصفح
# http://localhost:8000
```

### 📦 البناء والنشر
```bash
# بناء الموقع للنشر
npm run build

# تشغيل الإصدار المحسن محلياً
cd dist && python -m http.server 8000
```
